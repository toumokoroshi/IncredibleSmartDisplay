天気アイコン分類プレビュー

Meteocons の flat animated SVG を使った、14分類、補助分類、transition の全体一覧です。

14分類

Open-Meteo の weather code からまずこの分類へ落とし、基本アイコンを決めます。

補助分類

Yahoo!天気の「雨の可能性あり」「雷を伴う」「暴風雨/暴風雪」に相当する情報は、基本分類とは別の状態として重ねます。

transition 単体ルール

stable、then、occasional、temporary の4種類をすべて表示します。

transition 全組み合わせ

14分類を primary kind とし、同じ分類を除く13分類を secondaryKind にした全組み合わせです。 then、occasional、temporary の各パターンを生成しています。

then: のち

occasional: 時々

temporary: 一時

実装では全組み合わせを個別定義せず、kind、transition、secondaryKind から表示を合成するのがよいです。
